AutotunerPID Toolkit Release Notes

The AutotunerPID Toolkit 1.0 Release Notes describe the changes introduced in the latest version of the AutotunerPID Toolkit.


System Requirements

The AutotunerPID Toolkit requires the following MathWorks products:

  • MATLAB
  • Simulink
  • Control System Toolbox

The AutotunerPID Toolkit has been tested with the following platforms:

  • Windows - MATLAB v. 6.5 and above
  • Linux - MATLAB v. 6.5 and above
